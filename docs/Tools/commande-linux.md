# Liste de commande linux : 
---
| ls                        | afficher les fichier du dossier où on se trouve                                       |
| ------------------------- | ------------------------------------------------------------------------------------- |
| cd                        | changer de directory                                                                  |
| pwd                       | afficher le répertoire de travail                                                     |
| alias                     | permet de raccourcir des commandes                                                    |
| cp filetocopy  newfile    | permet de copier le fichier dans un autre                                             |
| rm                        | supprimer                                                                             |
| mv source_file  dest_file | déplacer un fichier ou renommer le fichier                                            |
| mkdir                     | crée un dossier                                                                       |
| touch                     | crée un fichier                                                                       |
| man                       | consulter le manuel                                                                   |
| chmod                     | donner des droits                                                                     |
| ./                        | exec un fichier                                                                       |
| sudo                      | droit admin                                                                           |
| nano                      | editer un fichier                                                                     |
| unzip                     | unzip un dossier                                                                      |
| cat                       | lire un fichier                                                                       |
| passwd                    | changer les mdp                                                                       |
| which                     | chemin complet  (which python)                                                        |
| less                      | inspecter les fichiers en avant et en arrière                                         |
| tail                      | affiche les 10 dernieres lignes                                                       |
| head                      | affiche les 10 premières lignes                                                       |
| grep                      | chercher du texte (grep "coucou" ) avec des flags (-c compter le nombre d'occurences) |
| whoami                    | affiche le nom d'user                                                                 |
| whatis                    | affiche un description                                                                |
| neofetch                  | affiche les info du système                                                           |
| wget                      | recup du contenu sur internet                                                         |
|                           |                                                                                       |
