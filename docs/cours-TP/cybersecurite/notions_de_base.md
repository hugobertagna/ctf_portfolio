
# Cours - Les bases de la cybersécurité 

## Définitions :
**L'Attaquant**: Une personne qui cherche à exploiter les vulnérabilités potentielles d'un système
**Virus** : Un code malveillant attachés à des fichiers exécutables.
**SSI** : La sécurité des systèmes d'information
**Cheval de Troie** : Malware exécutant des opérations malveillant sous couvert d'une fonction souhaité des utilisateurs et les envoyer (espion)
**Rootkit** : Ce malware permet de forcer l'accès administrateur sur un machine
**Vulnérabilité** : Faille matérielle ou logiciel qui permet d'avoir un accès non autorisé
**Menace** : La menace est une cause potentielle d'accident pouvant résulter en un dommage au système.
**Attaque** : Tentative volontaire d'i une ou plusieurs vulnérabilité.
**Intrusion** : Opération qui consiste  accéder aux autorisation aux donnée d'un système informatique ou d'un réseau. C'est la suite d'une attaque réussi.
**Cyberspace** : espace de communication constitué par l'interconnexion mondiale d'équipement , cela représente entre 18 milliards d'objets connecté 244 millions en France.
**Fishing** : Mail ou Message trompeur ( mail de phishing : faux mail d'impôt )

## Le rôle d'un étudiant en cybersécurité

- Protéger le systèmes d'informations
- Détecter des failles et les sécuriser

## Comment se protéger 

- Sécurité physique : sécuriser l'accès aux machines, au sein d'une entreprise, sécurisé  le SI envers ses employés
- Sécurité des communications : antivirus , Pare-feu
- Mise à jour automatiques
- Utilisateur avec des droits limités
- Les mots de passes : Utiliser un mot de passe différents pour chaque service, Utiliser un mot de passe suffisamment long et complexe, utiliser un gestionnaire de Mot de passe.
